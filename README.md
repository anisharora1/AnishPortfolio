## 🛠️ Built With
- **React.js** (Frontend library)
- **Vite** (Build tool)
- **Tailwind CSS** (Styling & responsive design)
- **Framer Motion** (Smooth animations and transitions)
- **Web3Forms** (Contact form email integration without a backend)

## ✨ Features
- **Hero Section:** Engaging introduction with animated typographics.
- **About/Skills:** Overview of my expertise and technical stack.
- **Projects Showcase:** Interactive gallery displaying my latest work.
- **Certificates Viewer:** Easily view my professional credentials in a fluid popup modal.
- **Blog Section:** Platform to share my thoughts and tutorials.
- **Contact Form:** Direct messaging to my inbox using Web3Forms.

## 💻 Running Locally

1. **Clone the repository:**
   ```bash
   git clone https://github.com/anisharora1/AnishPortfolio.git
   ```

2. **Navigate to the directory:**
   ```bash
   cd AnishPortfolio/frontend
   ```
   *(Note: Adjust the path if you pulled from the root folder)*

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Start the development server:**
   ```bash
   npm run dev
   ```
   Open `http://localhost:5173` in your browser to view the project.
